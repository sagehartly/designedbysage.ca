Event Poster

Design, Typography, Illustration

I made these fun, illustrative event posters for party nights hosted by a local bakery. I thought it would be fun to make an ongoing poster series where they all work together as a set but each has a different style as well.

Bright colours, a fun use of typography and interesting illustrations work to create a eye-catching set of posters that I hope to keep adding to in the future.
